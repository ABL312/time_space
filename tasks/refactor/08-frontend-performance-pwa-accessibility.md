## Owner
frontend-dev (@lisnshsjwkz)

## Goal
提升前端性能、PWA 可靠性、移动端可访问性。

## Scope
- 路由级 lazy loading，重组件 Map/AR 按需加载。
- 图片/媒体懒加载，占位骨架。
- PWA offline 提示与基础缓存策略确认。
- 可访问性：button aria、focus ring、对比度、触摸目标。
- 清理无意义动画，保留少量有功能的 motion。

## Acceptance
- npm run build 通过。
- 首屏无明显白屏，移动端交互不卡。
- 关键隐私 API（GPS/camera/gyro）失败时降级明确。

## Must follow
遇到报错/不确定/接口需要对接：立即停止并向 Tostar 汇报。禁止自行猜测绕过。
