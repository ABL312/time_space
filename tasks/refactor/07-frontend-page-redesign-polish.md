## Goal
按新 Design System 重绘核心页面，降低 AI 味，提高产品完成度。

## Scope
- Home：减少口号堆砌，突出地图入口/创建/推荐/个人数据。
- Map：地图为主，推荐/热力/附近胶囊为辅助抽屉。
- Capsule Detail：内容阅读优先，媒体/回应/收藏/分享结构清晰。
- Create：表单分步/分区，照片录音预览自然。
- Profile/Collections/Favorites：统一卡片和列表密度。
- AR/Fallback：移动优先，降级提示不突兀。

## Acceptance
- 移动端 375px 不溢出，主要操作明显。
- 每页 loading/empty/error/offline 状态齐全。
- npm run build 通过。
